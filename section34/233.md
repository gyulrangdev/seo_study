# SEO 이미지 최적화 실습 과제

## 현재 상태

```html
<img src="pot.jpg" />
```

## 최적화 목표

- 키워드: "요리 도구", "냄비"
- 4가지 최적화 요소:
  1. 이미지 태그
  2. alt 속성
  3. URL 구조
  4. 파일명

## 실습 과제

1. 기본 이미지 태그 개선

   - alt 텍스트 추가
   - title 속성 추가
   - 적절한 설명 포함

2. URL 및 파일명 최적화
   - 의미 있는 디렉토리 구조 설계
   - SEO 친화적인 파일명 설정

## 실습 방법

1. 현재 코드 분석
2. 필요한 개선사항 파악
3. 최적화된 코드 작성
4. 다음 영상에서 답안 확인 및 비교

## 참고사항

- 자연스러운 키워드 사용
- 과도한 키워드 스터핑 지양
- 사용자 경험 고려
