# 페이지 속도 최적화 연습 문제

## 주어진 페이지 정보

- 유튜브 비디오 포함
- 이미지 3개
- 텍스트 1,000단어
- CSS 라이브러리 3개
- JavaScript 라이브러리 4개
- 일반적인 블로그 포스트 형식

## 연습 과제

- 페이지 속도를 향상시킬 수 있는 8가지 방법을 찾아보기
- 가장 쉽고 기본적인 최적화부터 시작
- 정답은 8가지 이상이 될 수 있음
- 다음 영상에서 답안 공개 예정

## 문제 해결 접근 방식

- "낮은 곳의 과일"부터 수확하는 전략 사용
- 즉각적인 효과를 볼 수 있는 간단한 최적화부터 시작
- 복잡하고 어려운 최적화는 후순위로 배치

## 참고사항

- 개인별로 다른 해결책을 제시할 수 있음
- 하지만 기본적인 최적화 방법들은 공통적으로 포함되어야 함
- 실제 현장에서 적용 가능한 실용적인 해결책 찾기
